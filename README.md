# GameOne

Trabalho Colaborativo Autoral(TCA) - 05/12/2023

Este projeto foi desenvolvido em grupo como parte do trabalho de finalização do ano. O "GameOne" é um site que utiliza HTML, CSS, BootStrap, Javascript, PHP e MySQL.

## Pré-requisitos

Antes de iniciar, certifique-se de ter um servidor web configurado, como o [XAMPP](https://www.apachefriends.org/), ou outro de sua preferência.

## Como executar

1. Clone ou faça o download do repositório.
2. Configure o ambiente de servidor web (como XAMPP) em sua máquina.
3. Para configurar o banco de dados:
   - O arquivo de configuração do banco de dados está localizado em `GameOne/ADM/Model/conexao.php`.
   - As tabelas e dados necessários estão no script SQL: `GameOne/sql/tca2023.sql`. 
   - Importe o arquivo SQL no seu gerenciador de banco de dados, como o phpMyAdmin do xampp ou [MySQL Workbench](https://dev.mysql.com/downloads/workbench/).
4. Inicie o servidor e acesse o site pelo navegador.

## Estrutura Principal do Projeto

- **/ADM**: Contém as funcionalidades administrativas do site.
- **/CLIENTE**: Contém as funcionalidades administrativas do site.
- **/assets**: Contém as imagens/vídeos utilizadas no site 
- **/css**: Contém o css utilizado no site
- **/sql**: Contém o script sql + os BD's antigos 
- **/styles**: Contem mais uma parte do css utilizado no site
